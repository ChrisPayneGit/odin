Lists in HTML: Unordered (ul) & Ordered (ol)

Unordered List (Bullets):

  • This is an unordered list (ul).
  • Each element is tagged with "li"
  • This is the 3rd line of this unordered list
  • This list appears as bullet points

Ordered List (Numbered):

  1. This is an ordered list (ol).
  2. Each element is tagged with "li"
  3. This is the 3rd line of this ordered list
  4. This list appears as a numbered list
